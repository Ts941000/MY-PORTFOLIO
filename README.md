# Tejash Sharma | Premium Portfolio

![Portfolio Preview](./hero.png)

My personal developer portfolio built with modern web technologies. It features a clean, minimal black-and-white design with subtle interactive animations to showcase my projects, skills, and experience.

## ✨ Features

- **Interactive Cursor Blob:** A custom cursor that follows mouse movement with an inverted blend mode and expands on interactive elements.
- **Particle Network Background:** A dynamic, physics-based interactive background canvas.
- **3D Glassmorphism Cards:** Content cards that feature a smooth 3D tilt effect on hover, complete with dynamic under-glow.
- **Magnetic Buttons:** Navigation links and call-to-action buttons that organically pull towards the user's cursor.
- **Dynamic Scroll Animations:** Elements smoothly reveal themselves based on intersection observers.
- **Live LeetCode Stats:** Showcases real-time competitive programming statistics (Max Rating, Contests Attended).
- **Fully Responsive:** Perfectly optimized across desktop, tablet, and mobile devices with a custom off-canvas mobile menu.

## 🛠️ Tech Stack

- **HTML5:** Semantic architecture
- **Vanilla CSS3:** Advanced animations, CSS Grid/Flexbox layouts, CSS variables, and Glassmorphism effects
- **Vanilla JavaScript:** DOM manipulation, Intersection Observers, Custom physics for particle canvas, and cursor tracking.

## 🚀 Running Locally

Since this is a static website built entirely with Vanilla HTML, CSS, and JS, there is no complex build process required.

1. Clone the repository:
   ```bash
   git clone https://github.com/Ts941000/portfolio.git
   ```
2. Navigate into the directory:
   ```bash
   cd portfolio
   ```
3. Open `index.html` in your favorite browser, or use a tool like Live Server in VS Code for hot-reloading.
